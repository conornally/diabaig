
 
  Key Bindings:

  Note, player movement will ALWAYS be possible with the ARROW keys
  and using the NUMPAD, but here you can set the other keys.

 >[   ] - Move UP                      Reserved Keys:
  [   ] - Move DOWN                    --------------
  [   ] - Move LEFT                    1,2,3 - Casting spells
  [   ] - Move RIGHT                HJKLYUBN - Run directions 
  [   ] - Move UP+LEFT                 SPACE - Run 
  [   ] - Move UP+RIGHT                  >   - Descend Stairway
  [   ] - Move DOWN+LEFT                 <   - Ascend Stairway
  [   ] - Move DOWN+RIGHT                ?   - Show Help Info 
                                         Q   - Quit         
  [   ] - Eat                            S   - Save        
  [   ] - Drink                          .   - Skip Turn   
  [   ] - Read                           ,   - Pickup item 
  [   ] - Throw                          :   - Wizard mode Console
  [   ] - Apply Potion              
  [   ] - Fire Bow                  
  [   ] - Search Area                 
  [   ] - Show Inventory           
  [   ] - Rest until interupted
